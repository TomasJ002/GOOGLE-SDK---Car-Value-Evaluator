# GOOGLE-SDK---Car-Value-Evaluator
A Google SDK Java agent which evaluates the value of a set car based on the kilometers, age, condition and spec.
This is my pet project alongside the gas station price agent which is my and my dad´s personal everyday helper.
It takes the value of a car when new, and estimates it´s current value based on the kilometers driven, age of the car, the total condition of the car on the scale 1 (best) to 5 (worst), and the spec of the car, including extra addons.
